# 面试与复习

> **阶段：** 贯穿全程
> **前置条件：** 无（可在任意阶段参考使用）

## 模块简介

本模块提供按难度分级的 Python 面试题集和复习资料，帮助你高效准备面试。每道题都标注了考察知识点和关联模块，方便跳转复习。建议在完成每个学习阶段后，对应复习该阶段的面试题。

## 知识点列表

| 序号 | 知识点 | 描述 | 难度 | 推荐复习时机 |
|------|--------|------|------|-------------|
| 01 | [初级面试题](./01-basic/) | Python 基础语法、数据类型、控制流、函数、GIL、作用域 | ⭐ 初级 | 完成第一阶段后 |
| 02 | [中级面试题](./02-intermediate/) | 框架选型、ORM 优化、异步编程、任务队列、API 设计、部署 | ⭐⭐ 中级 | 完成第二、三阶段后 |
| 03 | [高级面试题](./03-advanced/) | GIL 深入、元类、描述符、内存管理、分布式架构、YOLO、RAG | ⭐⭐⭐ 高级 | 完成第四阶段后 |

## 各级别题目概览

### 01-初级面试题

| 题号 | 题目 | 考察知识点 |
|------|------|-----------|
| 1 | 基本数据类型与可变/不可变类型 | 数据类型、可变性 |
| 2 | `==` 和 `is` 的区别 | 对象比较、身份标识 |
| 3 | 列表推导式与 for 循环的区别 | 列表推导式、Pythonic 风格 |
| 4 | 装饰器原理与手写实现 | 装饰器、闭包、高阶函数 |
| 5 | GIL 及其对多线程的影响 | GIL、并发编程 |
| 6 | 深拷贝与浅拷贝 | 对象拷贝、引用语义 |
| 7 | `*args` 和 `**kwargs` | 函数参数、可变参数 |
| 8 | `with` 语句与上下文管理器 | 上下文管理器、资源管理 |
| 9 | Python 2 和 Python 3 的区别 | 版本差异 |
| 10 | 命名空间与 LEGB 作用域规则 | 作用域、命名空间 |
