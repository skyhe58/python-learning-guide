## ⚖️ 法律与道德规范声明

> **重要提醒：** 在进行任何网络爬虫操作前，请务必遵守以下规范：
>
> 1. **遵守 robots.txt 协议** — 在爬取任何网站前，先检查并遵守该网站的 robots.txt 文件规定
> 2. **遵守相关法律法规** — 不得爬取涉及个人隐私、商业机密或受版权保护的内容
> 3. **控制请求频率** — 避免对目标服务器造成过大压力，设置合理的请求间隔
> 4. **尊重网站服务条款** — 部分网站明确禁止爬虫行为，请遵守其服务条款
> 5. **仅用于学习和合法用途** — 本模块所有示例仅供学习参考，请勿用于非法目的
>
> 违反上述规范可能导致法律风险，请务必谨慎操作。

## 模块简介

本模块系统性地讲解 Python 网络爬虫技术，从 HTTP 基础到 Scrapy 框架，从静态页面到动态页面爬取，涵盖数据采集、清洗、存储的完整流程。所有示例均包含请求频率控制、异常处理和日志记录。

## 推荐学习顺序

1. **HTTP 基础与 requests** — 理解 HTTP 协议是爬虫的基础
2. **HTML 解析** — 掌握从网页中提取数据的核心技能
3. **数据存储** — 学会将爬取的数据持久化保存
4. **动态页面爬取** — 处理 JavaScript 渲染的页面
5. **Scrapy 框架** — 使用专业框架提升爬虫效率
6. **反爬策略与应对** — 了解常见反爬机制及应对方法
