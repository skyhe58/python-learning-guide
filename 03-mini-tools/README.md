# 小工具开发

> **阶段：** 第二阶段
> **前置条件：** [01-Python 基础](../01-python-basics/)

## 模块简介

本模块通过实战项目教你用 Python 开发各类实用小工具，包括文件批量处理、Excel/CSV 数据处理、PDF 操作、自动化脚本和命令行工具开发。每个工具都可以直接用于日常工作，提升效率。

## 知识点列表

| 序号 | 知识点 | 描述 | 难度 |
|------|--------|------|------|
| 01 | [文件批量处理](./01-file-batch/) | 文件批量重命名、格式转换、整理 | 入门 |
| 02 | [Excel/CSV 数据处理](./02-excel-csv/) | 使用 openpyxl/pandas 处理表格数据 | 入门 |
| 03 | [PDF 操作](./03-pdf-tools/) | PDF 合并、拆分、提取文本 | 入门 |
| 04 | [自动化脚本](./04-automation/) | 定时任务、系统操作自动化 | 进阶 |
| 05 | [命令行工具开发](./05-cli-tools/) | argparse 和 click 的用法与对比 | 入门 |

## 推荐学习顺序

1. **命令行工具开发** — 先掌握 CLI 开发基础，后续工具都会用到
2. **文件批量处理** — 最常见的自动化需求，入门简单
3. **Excel/CSV 数据处理** — 工作中最常用的数据处理场景
4. **PDF 操作** — 文档处理的实用技能
5. **自动化脚本** — 综合运用前面的知识，实现更复杂的自动化

## 模块依赖安装

```bash
pip install -r requirements.txt
```
