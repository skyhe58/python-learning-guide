# 常用功能

> **阶段：** 第二阶段
> **前置条件：** [01-Python 基础](../01-python-basics/)

## 模块简介

本模块涵盖 Python 开发中最常用的功能和库，包括正则表达式、日期时间处理、数据格式处理、数据库操作、HTTP API 开发、单元测试和日志管理。掌握这些技能后，你将能够应对大部分日常开发场景。

## 知识点列表

| 序号 | 知识点 | 描述 | 难度 |
|------|--------|------|------|
| 01 | [正则表达式](./01-regex/) | re 模块核心函数、常用模式、编译优化 | 入门 |
| 02 | [日期时间处理](./02-datetime/) | datetime/timedelta/timezone、格式化与解析 | 入门 |
| 03 | [数据格式处理](./03-data-formats/) | JSON/YAML/XML 的解析与生成、往返一致性 | 入门 |
| 04 | [数据库操作](./04-database/) | SQLite/MySQL/PostgreSQL 连接与操作、事务处理 | 进阶 |
| 05 | [HTTP API 开发](./05-http-api/) | FastAPI 和 Flask 核心概念与完整 REST API 示例 | 进阶 |
| 06 | [单元测试](./06-testing/) | pytest 框架、fixture、参数化测试、mock | 入门 |
| 07 | [日志管理](./07-logging/) | logging 模块配置、Handler/Formatter/Filter | 入门 |

## 推荐学习顺序

1. **正则表达式** — 文本处理的基础工具，应用广泛
2. **日期时间处理** — 日常开发中频繁使用
3. **数据格式处理** — JSON/YAML/XML 是数据交换的基础
4. **日志管理** — 良好的日志习惯应尽早养成
5. **单元测试** — 保证代码质量的关键技能
6. **数据库操作** — 大部分应用都需要数据持久化
7. **HTTP API 开发** — Web 开发的核心技能
