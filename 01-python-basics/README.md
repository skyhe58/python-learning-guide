## 模块简介

本模块面向有 Java 开发经验的程序员，通过与 Java 的对比来系统性地学习 Python 基础语法和核心概念。涵盖从环境搭建到面向对象编程的完整基础知识体系，每个知识点都配有可独立运行的代码示例，确保通过动手编码掌握技能。

## 推荐学习顺序

1. **环境搭建** — 搭建开发环境是一切的起点
2. **数据类型与变量** — 理解 Python 的类型系统是编程基础
3. **控制流** — 掌握条件判断和循环是编写逻辑的核心
4. **函数与装饰器** — 函数是代码复用的基本单元，装饰器是 Python 特色
5. **类与面向对象** — Java 开发者熟悉 OOP，对比学习效率最高
6. **模块与包管理** — 理解 Python 的模块系统，对标 Maven/Gradle
7. **异常处理** — 编写健壮代码的必备技能
8. **文件操作** — 实际工作中最常用的操作之一
9. **列表推导式与生成器** — Python 特色语法，提升代码简洁性
10. **Python 与 Java 核心差异总结** — 系统回顾，巩固理解
