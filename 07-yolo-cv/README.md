# 计算机视觉（YOLO）

> **阶段：** 第四阶段
> **前置条件：** [01-Python 基础](../01-python-basics/)，建议完成第二、三阶段后学习

## 模块简介

本模块讲解使用 Python 进行计算机视觉开发，从 OpenCV 图像基础到 YOLO 目标检测实战。涵盖图像处理、模型使用、模型训练和实际应用案例，帮助你快速上手计算机视觉项目。

## 知识点列表

| 序号 | 知识点 | 描述 | 难度 |
|------|--------|------|------|
| 01 | [OpenCV 图像基础](./01-opencv-basics/) | 图像读取/显示/保存、颜色空间转换、基础操作 | 入门 |
| 02 | [YOLO 模型介绍](./02-yolo-intro/) | YOLOv5/v8/v11 版本对比、特点与适用场景 | 入门 |
| 03 | [YOLO 目标检测实战](./03-yolo-detection/) | 使用 Ultralytics 库进行目标检测 | 进阶 |
| 04 | [YOLO 模型训练与微调](./04-yolo-training/) | 自定义数据集训练流程、标注格式 | 高级 |
| 05 | [图像预处理与后处理](./05-image-processing/) | 图像缩放、归一化、NMS 后处理 | 进阶 |
| 06 | [实际应用案例](./06-applications/) | 人脸检测、车辆检测、缺陷检测等场景 | 进阶 |

## 推荐学习顺序

1. **OpenCV 图像基础** — 图像处理的基础知识
2. **YOLO 模型介绍** — 了解 YOLO 系列模型的发展和特点
3. **YOLO 目标检测实战** — 动手使用预训练模型进行检测
4. **图像预处理与后处理** — 提升检测效果的关键技术
5. **YOLO 模型训练与微调** — 使用自定义数据集训练模型
6. **实际应用案例** — 了解真实场景中的应用
