## 概念说明

Matplotlib 是 Python 最经典的绘图库，支持折线图、柱状图、散点图、饼图等各种图表。在 AI 领域，它常用于数据可视化、训练过程监控（loss 曲线）和结果展示。

**在后续 AI 应用中的用途：**
- 数据探索性分析（EDA）
- 模型训练 loss/accuracy 曲线
- 检测结果可视化（标注框绘制）
